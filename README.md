# netsplit.uk

Static site built with Hugo.

## Prerequisites

* [Hugo (extended version)]*(https://gohugo.io/getting-started/installing/#fetch-from-github)

## Run the development webserver

With draft articles:

```
hugo serve -D
```

Without draft articles:

```
hugo serve
```

## Output to HTML

```
make build
```

## Clean up generated HTML

```
make clean
```
